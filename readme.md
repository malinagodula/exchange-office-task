# Exchange office task

This is training app: react + express + mongoDB + auth0

## Usage

Rename the .envexample to .env and add your MONGO_URI and AUTH0 credentials.

### Install dependencies

```
# Backend deps
cd backend
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

### Run Front

```
npm run dev
```
